## 计划与执行模式

Cline 会探索您的代码库并与您合作，在编写代码之前创建全面的计划，确保它理解您项目的完整上下文。

<Tabs>
  <Tab title="计划模式" icon="lightbulb">
    对于复杂任务，Cline 会探索您的代码库，提出澄清性问题，并在进行更改之前创建详细的实施计划。

    - 信息收集和上下文构建
    - 提出澄清性问题
    - 创建详细的执行计划
    - 与您讨论方法
  </Tab>

  <Tab title="执行模式" icon="play">
    一旦您批准了计划，Cline 将以完全的透明度和控制权实施解决方案。

    - 执行计划的操作
    - 使用工具修改文件和运行命令
    - 实施解决方案
    - 提供结果和完成反馈
  </Tab>
</Tabs>
