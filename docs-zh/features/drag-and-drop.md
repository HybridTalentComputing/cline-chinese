将文件拖放到 Cline 中是一种向对话添加图像、代码和其他文件的快速方法。

<Frame>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/dran-n-drop.gif"
		alt="Dragging and dropping files into Cline chat"
	/>
</Frame>

<Note>由于 VS Code 的特性，要将文件拖放到 Cline 聊天输入中，你需要拖动时按住 `Shift`。</Note>

将工作区文件拖放到 Cline 中将自动创建[文件提及](/features/at-mentions/file-mentions)。这允许你在对话中引用文件，而无需输入路径。

### 从 Finder/文件资源管理器拖动

你可以直接从系统的文件管理器将文件拖动到 Cline 中：

<Frame>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/drag-n-drop-finder.gif"
		alt="Dragging files from Finder into Cline"
	/>
</Frame>

### 支持的文件类型

Cline 支持从文件系统拖动外部图像、PDF、CSV、Excel 和其他文本文件，以及工作区中的文件。
