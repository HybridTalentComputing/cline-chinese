Cline 提供了管理你任务历史的工具，帮助你高效地组织、搜索和维护工作区。随着你随时间积累任务，这些功能对生产力变得至关重要。

## 访问任务历史

了解在 Cline 中打开和导航到任务历史的不同方式。无论你喜欢单击按钮、使用键盘快捷键还是命令面板，都有多种方便的方法来访问你过去的工作。

你可以通过以下方式访问任务历史：

1. **单击** Cline 侧边栏中的"历史"按钮
2. **使用命令面板**：搜索"Cline：显示任务历史"
3. **键盘快捷键**（如果在你的 VSCode 设置中配置）

## 任务历史界面

探索显示和管理所有任务的主要界面。本节涵盖布局、搜索功能、排序选项和过滤工具，这些工具帮助你高效地浏览累积的任务。任务历史视图为管理所有过去和当前的任务提供了全面的界面。

### 搜索和过滤

历史视图包括搜索和过滤功能：

#### 搜索栏
- 在所有任务内容中进行**模糊搜索**
- 搜索提示词、响应和代码
- 键入时即时过滤结果
- 在结果中高亮匹配的文本

#### 排序选项
按以下方式对任务进行排序：
- **最新**（默认）- 最近的任务在前
- **最旧**- 最早的任务在前
- **最昂贵**- API 成本最高的任务
- **最多 Token**- Token 使用率最高的任务
- **最相关**- 搜索时最佳匹配

#### 收藏夹过滤器
- 切换以仅显示已加星标的任务
- 快速访问你最重要的工作
- 与搜索结合进行精确过滤

## 任务操作

发现你可以在历史记录中对单个任务执行的各种操作。从重新打开和恢复任务到导出和管理任务，本节解释了任务操作的所有可用操作。

历史记录中的每个任务提供几个操作：

### 主要操作

- **打开**：单击任务以在 Cline 聊天中重新打开它
- **恢复**：从中断的地方继续中断的任务
- **导出**：将对话保存为 markdown 以便文档记录

### 管理操作

- **收藏** ⭐：单击星标图标以标记重要任务
- **删除** 🗑️：删除单个任务（收藏夹受保护）
- **复制**：基于现有任务创建新任务

## ⭐ 任务收藏夹

掌握收藏夹系统以标记和保护你最宝贵的任务。此功能允许你为重要工作加星标，防止意外删除，同时提供快速访问以参考实现和成功的模式。

收藏夹系统帮助你保留并快速访问重要任务。

### 使用收藏夹

**标记收藏夹**
- 单击任何任务旁边的星标图标
- 收藏夹时星标填充
- 再次单击以取消收藏

**保护功能**
- 收藏夹的任务受保护，不会意外删除
- 批量删除操作默认跳过收藏夹
- 可以通过明确确认覆盖保护

**收藏夹的使用案例**
- 参考你想要保留的实现
- 成功的问题解决模式
- 具有可重用代码片段的任务
- 重要的项目里程碑
- 团队成员的学习示例

## 任务指标

通过任务指标了解你的 Cline 使用情况。本节解释了如何跟踪 Token 使用情况、API 成本和其他指标，以帮助你优化工作流并有效管理资源。

了解你的任务指标有助于优化使用：

### 可用指标

- **Token 使用情况**：消耗的输入/输出 Token 总数
- **API 成本**：基于模型定价的估算成本
- **检查点数量**：创建的文件快照数量

### 使用指标

- **预算跟踪**：监控跨任务的 API 成本
- **效率分析**：识别昂贵的操作
- **模型比较**：比较不同模型之间的成本
- **优化**：找到可以更高效的任务
