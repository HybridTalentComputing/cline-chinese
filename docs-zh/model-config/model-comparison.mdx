## 性能比较

### 速度与质量权衡

| 优先级 | 推荐模型 | 原因 |
|----------|------------------|-----|
| **速度** | Cerebras 上的 Qwen3 Coder | 可用的最快推理 |
| **质量** | Claude Sonnet 4.5 | 对复杂任务最可靠 |
| **平衡** | DeepSeek V3 | 低成本的良好质量 |

### 工具可靠性

按工具使用可靠性排序的模型：
1. **Claude Sonnet 4.5** - 最可靠的工具执行
2. **GPT-5** - 优秀但偶尔有格式问题
3. **Gemini 2.5 Pro** - 适合标准工具
4. **DeepSeek V3** - 对基本工具可靠
5. **Qwen3 变体** - 复杂工具可能需要重试

## 提供商比较

### 提供商功能

| 提供商 | 可用模型 | 计费 | API 稳定性 | 支持 |
|----------|-----------------|---------|---------------|---------|
| **Cline** | 多个 | 基于积分 | 高 | 应用内 |
| **Anthropic** | 仅 Claude | 基于使用量 | 高 | 邮件 |
| **OpenRouter** | 100+ 个模型 | 基于使用量 | 高 | Discord |
| **OpenAI** | 仅 GPT | 基于使用量 | 高 | 论坛 |
| **本地** | 开源 | 免费 | N/A | 社区 |

### 提供商选择指南

根据以下选择提供商：
- **简单性**：Cline（无需 API 密钥管理）
- **多样性**：OpenRouter（访问所有模型）
- **直接访问**：单个提供商（Anthropic、OpenAI）
- **隐私**：Ollama 或 LM Studio（本地模型）
