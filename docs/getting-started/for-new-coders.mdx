> 💡 **提示：** 如果您完全不懂编程，请慢慢来，一步一步地学习。不要着急——Cline 会在这里指导您！

### 🚀 开始使用

在开始编程之前，请确保您已准备好这些必需品：

#### 1. **VS Code**

一个流行、免费且功能强大的代码编辑器。

-   [<u>下载 VS Code</u>](https://code.visualstudio.com/)

📺 **推荐的 YouTube 教程：** [<u>如何安装 VS Code</u>](https://www.youtube.com/watch?v=MlIzFUI1QGA)

> ✅ **专业提示：** 将 VS Code 安装在您的应用程序文件夹（macOS）或程序文件（Windows）中，以便从您的 dock 或开始菜单轻松访问。

#### 2. **组织您的项目**

在您的文档文件夹中创建一个名为 `Cline` 的专用文件夹，用于存放所有编程项目：

-   **macOS：** `/Users/[您的用户名]/Documents/Cline`
-   **Windows：** `C:\Users\[您的用户名]\Documents\Cline`

在您的 `Cline` 文件夹内，清晰地组织项目：

-   `Documents/Cline/workout-app` _（例如，健身跟踪应用）_
-   `Documents/Cline/portfolio-website` _（例如，展示您的工作）_

> 💡 **提示：** 从一开始就保持项目组织有序，这将为您节省时间并避免以后的混乱！

#### 3. **安装 Cline VS Code 扩展**

通过在 VS Code 中直接安装 Cline 扩展来增强您的编程工作流程：

-   开始使用 Cline 扩展教程

📺 **推荐的 YouTube 教程：** [<u>如何在 VS Code 中安装扩展</u>](https://www.youtube.com/watch?v=E7trgwZa-mk)

> ✅ **专业提示：** 安装后，重新加载 VS Code 以确保扩展正确激活。

#### 4. **基本开发工具**

高效编程所需的基本软件：

-   Homebrew (macOS)
-   Node.js
-   Git

👉 [<u>按照我们的详细指南，在 Cline 的逐步帮助下安装基本开发工具。</u>](https://docs.cline.bot/getting-started/installing-dev-essentials#installing-dev-essentials)

-   **对于 macOS：**
    -   [<u>在 Mac 上安装 Homebrew</u>](https://www.youtube.com/watch?v=hwGNgVbqasc)
    -   [<u>在 macOS 2024 上安装 Git</u>](https://www.youtube.com/watch?v=B4qsvQ5IqWk)
    -   [<u>在 Mac 上安装 Node.js (M1 | M2 | M3)</u>](https://www.youtube.com/watch?v=I8H4wolRFBk)

> ⚠️ **注意：** 如果在安装过程中遇到权限问题，请尝试以管理员身份运行您的终端或命令提示符。

🎉 您已经准备就绪！开始使用 **Cline** 更智能、更快速地编程吧。
