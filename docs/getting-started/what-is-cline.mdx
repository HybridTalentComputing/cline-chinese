---
title: "什么是 Cline？"
description: "Cline 简介，您在 VS Code 中的 AI 驱动开发助手。"
---

Cline 是一个与 Microsoft Visual Studio Code 集成的 AI 开发助手。它在您的 IDE 和 LLM 之间提供接口，促进代码开发，提高生产力，并降低新程序员的学习门槛。根据权限设置，Cline 可以读取/写入文件、执行命令、使用您的网络浏览器，并通过模型上下文协议服务器扩展其功能。

Cline 的独特之处在于其对代码生成的深思熟虑方法和广泛的集成能力。Cline 不仅仅是生成代码片段，而是通过与开发者协作，逐步规划解决方案，保持对整个开发环境的认知，并要求对所有更改进行明确批准。它能够理解大型代码库，加速新工程师的入职，并通过其模型上下文协议市场连接数百种工具，实现从简化项目部署到自动化事件响应的所有功能——所有这些都通过自然语言命令完成。
