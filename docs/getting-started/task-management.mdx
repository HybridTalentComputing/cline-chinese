# 任务管理

当您使用 Cline 时，您会随着时间的推移积累许多任务。任务管理系统帮助您组织、过滤、搜索和清理任务历史，以保持工作区高效。

## 访问任务历史

您可以通过以下方式访问任务历史：

1. 点击 Cline 侧边栏中的"历史"按钮
2. 使用命令面板搜索"Cline: 显示任务历史"

## 任务历史功能

任务历史视图提供了几个强大的功能：

### 搜索和过滤

-   **搜索栏**：使用顶部的模糊搜索快速按内容查找任务
-   **排序选项**：按以下方式排序任务：
    -   最新（默认）
    -   最旧
    -   最昂贵（最高 API 成本）
    -   最多令牌（最高令牌使用量）
    -   最相关（搜索时）
-   **收藏夹过滤器**：切换以仅显示收藏的任务

### 任务操作

历史视图中的每个任务都有几个可用操作：

-   **打开**：点击任务以在 Cline 聊天中重新打开它
-   **收藏**：点击星形图标将任务标记为收藏
-   **删除**：删除单个任务（收藏夹受保护，不会被删除）
-   **导出**：将任务的对话导出为 markdown

## ⭐ 任务收藏夹

收藏夹功能允许您标记重要任务，以便保存和快速找到。

### 收藏夹如何工作

-   **标记收藏**：点击任何任务旁边的星形图标以切换其收藏状态
-   **保护**：收藏的任务受保护，不会被单个和批量删除操作删除（可以覆盖）
-   **过滤**：使用收藏夹过滤器快速访问您的重要任务

## 批量操作

任务历史视图支持几个批量操作：

-   **选择多个**：使用复选框选择多个任务
-   **全选/全不选**：快速选择或取消选择所有任务
-   **删除所选**：删除所有选中的任务
-   **删除全部**：从历史中删除所有任务（收藏夹会被保留，除非您选择包含它们）

## 最佳实践

1. **收藏重要任务**：将参考任务或经常访问的对话标记为收藏
2. **定期清理**：定期删除旧或未使用的任务以提高性能
3. **使用搜索**：利用模糊搜索快速找到特定对话
4. **导出有价值的任务**：将重要任务导出为 markdown 以供外部参考

任务管理帮助您在 Cline 中保持有组织的工作流程，允许您快速找到过去的对话、保存重要工作，并保持历史记录清洁高效。
