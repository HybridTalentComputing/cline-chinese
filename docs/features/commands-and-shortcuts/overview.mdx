### 什么是编辑器集成？

编辑器集成是让您在工作时直接使用 Cline 的命令和快捷键。您无需先切换到 Cline 面板，而是可以选择代码，右键点击，并立即将其发送给 Cline 寻求帮助。
这些集成在 VSCode 的不同地方出现：

-   在编辑器上下文菜单（右键菜单）中 - "添加到 Cline"
-   在终端上下文菜单中 - "添加到 Cline"
-   在源代码控制视图中 - "生成提交消息"
-   作为键盘快捷键 - 各种 Cline 命令
-   作为快速修复选项（灯泡菜单） - "用 Cline 修复"、"用 Cline 解释"、"用 Cline 改进"

### 它们如何工作

当您使用这些命令时，Cline：

-   捕获相关上下文（选中的代码、文件路径、终端输出等）
-   聚焦 Cline 界面
-   使用捕获的上下文创建对话
-   在某些情况下，自动生成建议的提示

在幕后，这些命令使用 VSCode 的扩展 API 来注册命令、访问编辑器状态和控制 VSCode 的界面。
