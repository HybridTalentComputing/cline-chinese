将文件拖放到 Cline 中是向对话添加图像、代码和其他文件的快速方法。

<Note>由于 VS Code 的特殊性，要将文件拖放到 Cline 聊天输入中，您需要在拖放时按住 `Shift` 键。</Note>

将工作区文件拖放到 Cline 中将自动创建[文件提及](/features/at-mentions/file-mentions)。这允许您在对话中引用文件，而无需输入路径。

### 支持的文件类型

Cline 支持从文件系统拖放外部图像、PDF、CSV、Excel 和其他文本文件，以及工作区中的文件。
